Descripción del Proyecto

El proyecto se centró en centralizar la gestión de requerimientos, permitiendo una comunicación fluida y un seguimiento en tiempo real de cada solicitud de servicio. Se implementó una arquitectura robusta para garantizar que la asignación de tareas sea rápida, segura y transparente para ambas partes.

Desafíos Técnicos

Uno de los mayores retos fue la integración de la base de datos con el frontend dinámico, garantizando tiempos de respuesta menores a 200ms.

  • Optimización de consultas SQL.
  • Implementación de seguridad por capas.
  • Interfaz responsive compatible con dispositivos móviles.
