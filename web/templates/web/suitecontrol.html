SuiteControl

La solución definitiva para la administración eficiente y modernización de Hoteles y Moteles.

  • Control Total: Gestión de estados de habitaciones (Disponible, Ocupada, Limpieza).
  • Finanzas Pro: Reportes de ingresos detallados por turnos y filtros de fechas.
  • Arquitectura Sólida: Base de datos optimizada en SQLite para máxima velocidad y portabilidad.
  • Rendimiento Java: Desarrollado en Java Swing para un funcionamiento estable en escritorio.

Requisitos del Sistema

SuiteControl está optimizado para ofrecer el mejor rendimiento en entornos Windows modernos.

Windows 11

Recomendado
  • Sistema Operativo: Windows 10 o Windows 11 (64-bit).
  • Procesador: Intel Core i3 / AMD Ryzen 3 o superior.
  • Memoria RAM: 4GB mínimo (8GB para multitarea fluida).
  • Entorno: Java Runtime Environment (JRE) 8 o superior.
  • Disco: 250MB de espacio libre en SSD/HDD.

Nota: Si utilizas 7-Zip para extraer el instalador, asegúrate de ejecutar el archivo con permisos de administrador la primera vez.
